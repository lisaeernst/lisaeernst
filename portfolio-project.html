Project Breakdown

Bootstrap was a lof of fun to get to know. Still more to learn...

This was my final, and ninth project, for Treehouse. We were asked to create a portfolio for ourselves which included showcasing the highlighted projects created during the course. We were introduced to Bootstrap in Unit 9. I could see this was an important tool to learn so I created this site using this framework.

I admit, it was a bit of a mind bender coming from hand coding with the mindset of creating lean code in a separate style sheet to calling your main styles and layout in the html document. But, for a first project I was pleased with the end result given these parameters for project 9.

Portfolio Project for Unit 9

My last project for Treehouse

Using the AOS library for interaction

Fading sections left and right into view as the user scrolls

I can see how quickly a site can come together using Bootstrap, but on the other side I can also see the challenges it can present especially when starting out. Because of this I supplemented my learning by taking a focused course on Bootstrap on the Udemy platform while developing this site. This helped me to get another point of view in the learning process and to see a few different projects/techniques created in the framework.

I added the AOS (Animate on Scroll) library to this project as I liked the fade in’s available with it. I also used the jsdelivr.js smooth scroll plugin because I am visually jarred when a site quickly flashes down a longer page to a spot when following a link. I am not a designer who puts a lot of animation and or effects into my sites just because I can do it. I myself can find it distracting and sometimes dizzy making. But I do enjoy the focus points that some animations create for the end user, and I felt these were a couple of them.

I used Photoshop to create the images for the pre-made mockups. I wanted to allow my viewer to see the real world application of my projects such as viewed on an iPhone, or on a desktop etc. I enjoy this depth when I see other designers using these visual tools as I feel it draws the viewer in to another world they can visualize themselves in.

In the end I exceeded expectations on this project which was great. I haven’t designed with this pink hued colour palette before but it just seemed to fit this project as I wanted something a bit more laid back and soothing after such a rigorous course of learning.

I am enjoying the current design styles that we see online. Focusing on typography such as size and font pairings, clean layouts, imagery, and the generous use of white space are creating such a sophisticated and bold new approach. It really is exciting to see, and be part of. This project helped me explore some new techniques, both technically and visually.

My design aesthetic has always been clean and uncluttered. Being a visual artist as well, and one that loves colour, I want to incorporate more colour theory and creative layout ideas to my future designs. I want to implement more magazine/print layout principles into my designs in the future as well. Thankfully we have CSS Grid and Flexbox to assist with these layout challenges and for fast prototyping, Bootstrap.

There are so many amazing designers out there to learn from. Everyday I am searching, and soaking up, new ways to create.
