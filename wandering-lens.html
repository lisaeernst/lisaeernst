Project Breakdown

Helping you find the images you need fast

This was my fifth project for Treehouse. It is an interactive photo gallery with a working search filter that filters through all of the images on the site and displays only the images that match the search criteria (any of the words included in the caption text).

It was my first foray into implementing Javascript into a Treehouse project and it helped me to begin understanding how to maniplulate html DOM elements using JS.

The site also has a Lightbox feature using the baguetteBox.js script incorporated into it allowing the end user to scroll through the images full screen along with a caption located at the bottom of the image. This one was a challenge as it was the first time I had to implement Javascript into a project for this course.

And an even wittier subheading.

I had some fun adding a more modern feel to this site by adding Google fonts and a header section just to modernize the look a bit more. I like minimalist design and one of my favorite things about being a designer is how simple changes can completly change the mood and elevate the users experience.

When I submitted this site for grading the grader wrote back saying that my custom styles were super neat (their words) and very professional looking especially the header. I was so pleased they liked it.

I coded the site using html and css and css grid for layout. The site is responsive without using media queries. I then installed the baguetteBox.js script as per the documentation on the GitHub page. This was fairly easy to do and it’s a very effective solution.

The more difficult part for me was coding the custom search filter in Javascript. I wanted to go the extra mile and learn how to create this filter on my own and not use a plugin. I admit, I did end up asking for some help but I got it done and made sure I understood how everything worked each step of the way.
